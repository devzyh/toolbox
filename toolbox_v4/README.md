## DZ Tools
* 基于[it-tools](https://github.com/CorentinTh/it-tools)定制的工具箱
* 移除部分工具，添加自定义工具

## 项目运行
### 依赖安装

```sh
pnpm install
```

### 运行开发环境

```sh
pnpm dev
```

### 构建生产环境

```sh
pnpm build
```

### 代码格式化

```sh
pnpm lint
```

### 创建新工具

```sh
pnpm run script:create:tool my-tool-name
```
